How this website has been built?

This website was built using contents from the Wiktionary and WikiSource.

I'm extracting words equivalences from the wiktionary dump.
This is not a hard task, the script looks quite similar than the scripts that you can find on this page.

The bilingual books were made from public domain books found on WikiSource.
Making one bilingual book usually require me about one day.
This current version of the web site currently uses 9 bilingual books.
I'm not using an automatic program to parallelise the books like Andras Farkas, I'm just using a couple of scripts like this one.
Then the pages content are generated using this small script.
